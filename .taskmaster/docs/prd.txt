# Moonrush - Solana Token Creator Platform PRD

## Executive Summary

Moonrush is a professional web platform for creating SPL tokens on the Solana blockchain with no coding required. The platform enables users to launch tokens in under 5 minutes with advanced features like custom metadata, authority revocation, and premium creator options. Built with Next.js, TypeScript, and advanced Web3 integrations.

## Product Vision

To democratize token creation on Solana by providing the most user-friendly, secure, and feature-rich token launcher that enables anyone to create professional-grade SPL tokens without technical expertise.

## Current Platform Features

### Core Token Creation
- SPL Token Minting with Solana Web3.js integration
- Custom Metadata: name, symbol, decimals, supply, description, image upload
- IPFS Integration: dual-mode storage (test/production)
- Authority Management: revoke mint, freeze, and update authorities

### Wallet Integration
- Multi-Wallet Support: Phantom, Solflare, Coinbase wallet adapters
- Secure Transaction Handling: native Phantom signAndSendTransaction API
- Wallet Detection and optimization

### User Interface
- Landing Page with moon-themed branding
- Dashboard with user-specific token management
- Token Creation Form with real-time cost calculation
- Responsive Design with TailwindCSS

### Premium Features
- Custom Creator: set custom creator address in metadata
- Authority Revocation: enhanced decentralization
- Social Links Integration: Twitter, Telegram, website links
- Token Cloning: clone existing tokens from trending lists

### Data Management
- SQLite Database: token creation logging and persistence
- User Context: session management and token history
- API Endpoints: RESTful API for data operations

### Pricing Structure
- Base Token Creation: 0.1 SOL
- Premium Features: +0.1 SOL each
- Maximum Cost: 0.5 SOL for all features
